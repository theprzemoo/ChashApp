# ChashApp Brand Guide

## Core direction

- Visual style: dark, clean, security-first, modern desktop utility.
- Mood: premium but understandable for non-technical users.
- Primary accent: mint-cyan.
- Secondary accent: steel-blue.

## Suggested palette

- Background: `#08111B`
- Surface: `#102031`
- Surface elevated: `#142332`
- Border: `#294357`
- Primary accent: `#2DF0D0`
- Primary text: `#F4FBFF`
- Muted text: `#9FB5C7`
- Success: `#4ADE80`
- Warning: `#FACC15`
- Danger: `#F87171`

## Typography

- UI font: Inter
- Monospace support: Consolas / Cascadia Mono fallback where needed

## Release branding checklist

- Add app icon `.ico` for Windows publish.
- Add installer banner if using Inno Setup customization.
- Capture a real screenshot after first successful local build.
- Use the same accent and spacing system across README and repository graphics.
